MES PROJETS

Site Web Dracovente

Dans le cadre de notre apprentissage des langages HTML et CSS, nous avons dû réaliser un projet à faire sur un site web de vente d’un produit de notre choix. Nous avons alors choisi le Pokémon Dracaufeu comme produit de vente. Nous avons donc pu montrer nos compétences en HTML et en CSS en créant ce site avec une équipe de quatre développeurs. Il nous a aussi permis de nous entrainer sur l’utilisation de Git pour écrire le code du projet.

Le site Web est responsive, mais les formulaires ne sont pas fonctionnels.
